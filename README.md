# 🎬 Trailer Hub

A modern, cinematic movie and TV show discovery platform powered by the TMDB API.

## ✨ Features

- 🔍 **Smart Search** — Search millions of movies and TV shows with real-time autocomplete
- 🎬 **Inline Trailer Player** — Watch trailers embedded directly on the page, no redirects
- 📈 **Daily Trending** — Live trending movies and shows updated every day from real stats
- 🎭 **Genre Browsing** — Browse by Action, Comedy, Horror, Sci-Fi and 10+ more genres
- 🔞 **Age Rating Filter** — Filter by G, PG, PG-13, R ratings
- 🔖 **Watchlist** — Save movies to your personal watchlist (persisted in localStorage)
- 🕐 **Watch Later** — Separate watch later list saved to your browser
- ⭐ **Star Ratings** — Rate any movie 1–5 stars, saved locally
- 🕘 **Recently Viewed** — Auto-tracks your last 20 opened movies
- ♾️ **Infinite Scroll** — Results load automatically as you scroll
- 🌙 **Dark / Light Mode** — Toggle with one click, preference saved
- 📱 **PWA** — Installable as a mobile or desktop app
- ⌨️ **Keyboard Navigation** — Arrow keys, Enter, and / shortcut
- 🔗 **URL State** — Search results reflected in the URL for sharing
- 🎥 **Movie Detail Page** — Full cast, director, runtime, budget, similar movies

## 🛠️ Built With

- **HTML5 / CSS3 / Vanilla JavaScript** — No frameworks
- **TMDB API** — Movie and TV show data
- **localStorage** — Watchlist, ratings, recently viewed
- **Service Worker** — Offline support and caching
- **CSS Custom Properties** — Full dark/light theming

## 🔧 Setup

1. Clone the repo
```bash
   git clone https://github.com/alifahes-fhs/trailer-hub.git
```
2. Get a free API key from [themoviedb.org](https://www.themoviedb.org/settings/api)
3. Open `script.js` and replace `API_KEY` with your key
4. Open `index.html` with Live Server or any local server
